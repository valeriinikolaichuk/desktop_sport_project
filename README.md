## About the Project
This is a study project developed using **C++** and **Qt Framework**. 
The goal was to build a functional desktop application while mastering:
*   Qt Widgets and Layouts
*   Signal & Slot mechanism
*   C++ memory management in Qt
*   Cross-platform compilation

## Tech Stack
*   **Language:** C++
*   **Framework:** Qt 6.10.2
*   **Compiler:** MSVC 2022 (x86_64)
*   **Build System:** CMake (recommended version 3.16+)
